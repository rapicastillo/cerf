Media: Smugmug provides the following emfield modules
 - Media: Smugmug Image
 - Media: Smugmug Video
 - Media: Smugmug Gallery

Installation Instructions

Enable "Embedded Media Field Module"
Enable "Embedded Image Field" Module and "Media: Smugmug Image" to support embedding images from smugmug.
Enable "Embedded Video Field" Module and "Media: Smugmug Video" to support embedding videos from smugmug.
Enable "Embedded Video Field" Module and "Media: Smugmug Gallery" to support embedding gallerys from smugmug.

See the README.txt for further information.
